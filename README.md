# Miss Universe Philippines 2026 Scoring System

A clean professional React scoring dashboard.

## Scoring flow

1. Score all 31 candidates for:
   - Swimsuit
   - Evening Gown
2. Click **Submit Top 15 Scores**
3. The system verifies and reveals the **Top 15**
4. Q&A scores unlock only for the Top 15
5. Enter Q&A scores
6. Click **Submit Final Q&A**
7. The system reveals the **Top 7 Titleholders**

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

This project already includes a GitHub Actions workflow.

1. Upload all files to a GitHub repository.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, choose **GitHub Actions**.
4. Push to the `main` branch.
5. Wait for the deployment action to finish.
